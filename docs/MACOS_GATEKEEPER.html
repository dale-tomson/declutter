macOS "Not Opened" Warning

✅ Declutter is completely safe. This warning appears because the application is not notarized by Apple, which requires a paid Apple Developer account.

Why Does This Happen?

macOS includes a security feature called Gatekeeper that blocks applications from unidentified developers. Since Declutter is a free, open-source project, we currently do not have a paid Apple Developer certificate to sign the application.

Solution: The "Right-Click" Workaround

You only need to do this once when you first open the application:

  1. Locate the App: Open Finder and find the Declutter app.
  2. Right-Click: Right-click (or Control-click) on the Declutter icon.
  3. Select Open: Choose Open from the context menu.
  4. Confirm: A dialog box will appear asking if you're sure. Click Open.
Note: If you try to double-click the app without doing the right-click step first, you will simply see a message saying the app "cannot be opened". You must use the right-click menu to see the "Open" option.
