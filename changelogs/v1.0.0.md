### Core Functionality
- **Auto Organization** - Automatically sorts files into Year/Month folders (e.g., `2024/03-March/`)
- **Timestamp-based Sorting** - Uses file modification dates to determine the correct folder
- **Safe Operations** - Skips files that already exist at the destination to prevent overwrites

### User Interface
- Modern, clean UI built with Fyne
- Custom color theme matching the brand
- Real-time activity log showing file operations
- Progress indicator during organization
- Confirmation dialog before organizing files

### Platform Support
- Windows (amd64)
- macOS (amd64, arm64)
- Linux (amd64, arm64)

### Additional
- Custom application icon
- GitHub Pages website with automatic OS detection for downloads
